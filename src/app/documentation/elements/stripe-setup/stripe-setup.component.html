Set Up Payments

All payments for Lara-Commerce are set up through Stripe which is a third party payment gateway. A payment gateway is a merchant service that authorizes credit card or direct payments processing for an e-commerce site. One key advantage to using a payment gateway is that your customer's credit card information never touches your servers. Another important advantage is that stripe is PCI compliant.

Step 1: Create a Stripe Account

Visit Stripe and click Create an Account.

Step 2: Integrate Stripe with your System

In order to integrate Stripe with Lara-Commerce you need your Publishable Key and your Secret Key. To get both of your keys you need to be registered with stripe. Once you are registered visit the you Stripe Dashboard and on the side bar you should see a link called API. On this page you will see your Publishable Key and your Secret Key.

Now open your Lara-Commerce .env file that you created when you installed Lara-Commerce. In the .env add your Publishable Key and your Secret Key.

  • STRIPE_KEY=Publishable Key
  • STRIPE_SECRET=Secret Key

Step 3: Test that Lara-Commerce and Stripe was integrated

In order to confirm that stripe has been integrated into Lara-Commerce correctly just run the group test live_payment. Phpunit needs to be set up before the tests can be run. For a guide on that please visit set up Phpunit. To run tests in your terminal cd into your Lara-Commerce folder and run the phpunit --group live_payment command. This may take several minutes because each test connects to your stripe account. 

  phpunit --group live_payment

If Lara-Commerce was successfully integrated with stripe then phpunit should return green. For more information about stripe please visit the Stripe Documentation.
