Lara-Commerce uses two separate testing frameworks: PHPUnit and Laravel Dusk. PHPUnit tests the server end of Lara-Commerce while Laravel Dusk tests the browser.

PHPUnit

PHPUnit will test the php code and the http requests to the server. To run PHPUnit tests simple open your terminal and go to the Lara-Commerce folder. Then run phpunit. This will begin your test code. 

  phpunit

A common error is if you have not created an alias for PHPUnit. If you run PHPUnit and receive the following error bash: phpunit: command not found then simply run vendor/bin/phpunit.

There are a lot of tests in PHPUnit, which take a long time to run. It may not be possible to run every test on your system at once so you may want to break up running your tests into smaller units.

Unit Tests

Unit tests test the following values.

  • Error Tracking
  • Date Formatting
  • Shipping Date Estimator

To run PHPUnit unit test in your terminal run phpunit --group unit 

  PHPUnit --group unit

Integration Tests

Integrated tests the following items

  • Laravel Models: phpunit --group models
  • Database Classes: phpunit --group database
  • Laravel Transformers: phpunit --group transformers

To run PHPUnit integration tests in your terminal run phpunit --group unit 

  phpunit --group integration

Acceptance Tests

Acceptance tests the following items

  • http requests related to addresses: phpunit --group addresses
  • http requests related to the HomeController: phpunit --group home
  • http requests for login requestsphpunit --group login
  • http requests for registration requestsphpunit --group register
  • http requests for order requestsphpunit --group order
  • http requests for order stripe paymentsphpunit --group live_payment
  • http requests to view productsphpunit --group products
  • http requests for searching productsphpunit --group search
  • http requests for shopping cartphpunit --group shopping_cart
  • http requests for user accountsphpunit --group user_account

To run PHPUnit acceptance tests in your terminal run phpunit --group acceptance 

    phpunit --group integration

Laravel Dusk

Laravel Dusk provides browser automation to test Lara-Commerce in google chrome. For a full guide of how Laravel Dusk operates please visit the Laravel's Documentation. To run Laravel's Dusk tests simple open your terminal and go to the Lara-Commerce folder. Then run php artisan dusk. This will begin your test code.
