Installing Lara-Commerce

Step 1: Download the Application

Clone the git repository: 

      git clone https://github.com/shawndl/LaraCommerce.git

Enter the LaraCommerce director


      cd LaraCommerce

Step 2: Install all the Dependencies

Install all the composer requirements 

  composer install

Install all the npm requirements 

  npm install

Step 3: Set Up your local env file

In order for your project to work you need to create an .env file. The command php artisan publish:env will create an .env.example file for your project. Change the name of that file to .env before you can complete the next step. 

  php artisan publish:env

Step 4: Generate a application key

Next generate your app key with the following command 

  php artisan key:generate

Step 5: Migrate your database

For your next step to work you need to set up your local environment. If you need any help with that you can check out any of the following links. Of course, this will work with either MAMP, WAMP, or XAMPP.

  • Homestead: Laravel Homestead is an official, pre-packaged Vagrant box that provides you a wonderful development environment without requiring you to install PHP, a web server, and any other server software on your local machine.
  • Say Hello to Laravel Homestead 2.0: If you have any trouble setting up your environment. Please check out this video on setting up homestead from laracast.

In order to migrate your database you need to set in up first. To do that in homestead simple ssh into you server with the following command. 


        homestead ssh

Next enter your database by enter the following commands. 

  mysql -uhomestead -p

Then it will prompt you to enter your password. Once you are in mysql then create a database with the following command. 

    CREATE DATABASE LaraCommerce;

Before you can use your database with your project you need to register it in the .env file you created earlier. Open it up in the text editor of your choice and make sure the following values are set

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=LaraCommerce
DB_USERNAME=homestead
DB_PASSWORD=secret

Now you can add all the tables to your database by entering php artisan migrate. 

  php artisan migrate
Your project is now ready to be viewed in a website. If you want to enter some test data then enter php artisan db:seed 

      php artisan db:seed
