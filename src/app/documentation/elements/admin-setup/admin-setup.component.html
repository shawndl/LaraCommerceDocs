Set Up Admin Page

In order to manage the Lara-Commerce site some admin pages have been added. Admin pages manage the products, sales, taxes, categories, and orders of the site. It also allows the administrator to view user details such as current orders. In order to use the admin pages the logged in user must have a role as an admin. In order to create an admin please follow the steps below.

Step 1: Add an Admin to the Database

In the terminal window, cd into the Lara-Commerce folder and type the command laracommerce:admin. This command will add a new role in the database called 'admin'. 

  laracommerce:admin
