Features

Live Searches

Live searches are performed with Algolia, which is a third party system that allows your searches to be form more advanced by checking for spelling errors and allowing your products team to decided which products should display first.

Accepts Payments

Lara-Commerce accepts payments securely using Stripe. Stripe is a third party system that will process your users credit card information and handle all of the compliance issues. Your system will never see your customers credit card information.

Tested

This application has been tested with Phpunit and Laravel Dusk. Running these tests will help you determine if your system has been installed correctly or if any changes have created some code rot.

Process Orders

The system will assist your customers in completing their orders including

  • Editing the users cart
  • Selecting a shipping address
  • Accepting payments
